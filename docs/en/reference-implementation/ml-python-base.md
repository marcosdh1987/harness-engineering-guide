# ml-python-base

[`marcosdh1987/ml-python-base`](https://github.com/marcosdh1987/ml-python-base) is a public reference implementation.

It shows how a Python and ML template can organize AI rules, skills, adapters, automation, and documentation in a reusable repository structure.

Use it as a public example of layout and process. Do not assume private access, and do not copy private or sensitive content into this guide.
