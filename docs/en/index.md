# Harness Engineering Guide

Harness Engineering Guide is an educational resource for building durable systems around AI coding tools.

## What is Harness Engineering?

Harness Engineering focuses on the system around the model: rules, context, skills, adapters, workflows, validation gates, drift control, and documentation.

## Who this guide is for

- AI-assisted software engineers
- ML engineers
- Technical leads
- People building reusable AI coding workflows

## Reference implementation

[`marcosdh1987/ml-python-base`](https://github.com/marcosdh1987/ml-python-base) is a public reference implementation. It shows one way to organize AI rules, skills, adapters, automation, and documentation in a Python and ML-oriented repository.
