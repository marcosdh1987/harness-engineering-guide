# Learning Path

Use this sequence to study the guide with a practical progression.

1. **Core concepts** — understand Harness Engineering and the role of context.
2. **Rule architecture** — learn how governance, operational skills, automation, and orchestration fit together.
3. **Tool adapters** — see how different AI coding tools consume instructions and context.
4. **Patterns** — study reusable structures for working loops, rule layering, adapter design, and drift control.
5. **Practical labs** — create rules, create skills, validate changes, and publish safe documentation.
6. **Open-source readiness** — confirm that shared repositories are safe, clean, and reviewable.
