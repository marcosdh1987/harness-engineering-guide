# AI-Assisted Working Loop

A durable working loop helps AI-assisted changes stay focused.

1. Understand the task.
2. Plan the work.
3. Select the right rules and skills.
4. Make focused changes.
5. Run validation.
6. Review the diff.
7. Update documentation.
8. Sync generated artifacts if needed.

This loop reduces random edits and makes review easier for humans and tools.
