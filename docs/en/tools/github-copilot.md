# GitHub Copilot

## Role in a workflow

GitHub Copilot is useful for inline suggestions, chat-based assistance, and repository-aware development support.

## Useful context

Copilot benefits from readable repository docs, stable conventions, examples, and clear task framing near the code or documentation being changed.

## Harness adaptation

A harness should make local context easy to discover, keep instructions durable, and rely on automation to verify suggestions before they are merged.
