# OpenCode

## Role in a workflow

OpenCode can act as part of a flexible AI-assisted workflow for editing, exploration, and structured task execution.

## Useful context

A harness should assume the tool needs clear repository intent, practical examples, and easy access to validation steps.

## Harness adaptation

Treat the tool as a consumer of shared rules instead of a special case. Use adapters only when instruction shape or task packaging needs to differ.
