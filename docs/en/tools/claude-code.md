# Claude Code

## Role in a workflow

Claude Code can be used for repository exploration, focused edits, validation, and review-oriented tasks.

## Useful context

A harness should provide clear repository rules, practical task guidance, validation commands, and well-placed examples.

## Harness adaptation

Prefer concise durable instructions, explicit safety boundaries, and discoverable workflows. Keep the context organized so the tool can select the right files and checks without guessing.
