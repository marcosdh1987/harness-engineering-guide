# Ciclo de trabajo asistido por IA

Un ciclo de trabajo duradero ayuda a que los cambios asistidos por IA se mantengan enfocados.

1. Entender la tarea.
2. Planificar el trabajo.
3. Seleccionar las reglas y skills correctas.
4. Hacer cambios enfocados.
5. Ejecutar validación.
6. Revisar el diff.
7. Actualizar documentación.
8. Sincronizar artefactos generados si hace falta.

Este ciclo reduce ediciones aleatorias y facilita la revisión para personas y herramientas.
