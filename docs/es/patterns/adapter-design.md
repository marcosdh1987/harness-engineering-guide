# Diseño de adapters

Los adapters son necesarios porque las herramientas de programación con IA esperan instrucciones con formas diferentes.

## Por qué importan

- Una herramienta puede preferir archivos del repositorio.
- Otra puede preferir entradas estructuradas por tarea.
- Otra puede depender de configuración generada o command wrappers.

## Buena práctica

Genera o sincroniza adapters desde una fuente de verdad cuando sea posible. Evita mantener manualmente copias divergentes de políticas entre herramientas.
