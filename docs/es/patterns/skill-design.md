# Diseño de skills

Las skills duraderas deben ser fáciles de activar y fáciles de verificar.

## Lista de diseño

- Trigger claro
- Scope definido
- Inputs conocidos
- Outputs esperados
- Pasos de validación
- Ejemplos
- Failure modes

Una buena skill reduce ambigüedad sin fingir que todas las tareas son idénticas.
