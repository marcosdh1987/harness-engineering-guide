# Ruta de aprendizaje

Usa esta secuencia para estudiar la guía con una progresión práctica.

1. **Conceptos centrales** — entiende Harness Engineering y el rol del contexto.
2. **Arquitectura de reglas** — aprende cómo encajan gobernanza, skills operacionales, automatización y orquestación.
3. **Adapters de herramientas** — revisa cómo distintas herramientas de programación con IA consumen instrucciones y contexto.
4. **Patrones** — estudia estructuras reutilizables para ciclos de trabajo, capas de reglas, diseño de adapters y control de drift.
5. **Laboratorios prácticos** — crea reglas, crea skills, valida cambios y publica documentación segura.
6. **Preparación para open source** — confirma que los repositorios compartidos sean seguros, limpios y fáciles de revisar.
