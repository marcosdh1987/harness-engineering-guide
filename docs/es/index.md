# Harness Engineering Guide

Harness Engineering Guide es un recurso educativo para construir sistemas duraderos alrededor de herramientas de programación asistida por IA.

## ¿Qué es Harness Engineering?

Harness Engineering se enfoca en el sistema alrededor del modelo: reglas, contexto, skills, adapters, workflows, validation gates, drift control y documentación.

## ¿Para quién es esta guía?

- Ingenieros de software asistidos por IA
- Ingenieros de ML
- Líderes técnicos
- Personas que crean workflows reutilizables de programación con IA

## Implementación de referencia

[`marcosdh1987/ml-python-base`](https://github.com/marcosdh1987/ml-python-base) es una implementación de referencia pública. Muestra una forma de organizar reglas de IA, skills, adapters, automatización y documentación en un repositorio orientado a Python y ML.
