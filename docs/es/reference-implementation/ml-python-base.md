# ml-python-base

[`marcosdh1987/ml-python-base`](https://github.com/marcosdh1987/ml-python-base) es una implementación de referencia pública.

Muestra cómo una plantilla de Python y ML puede organizar reglas de IA, skills, adapters, automatización y documentación dentro de una estructura reutilizable de repositorio.

Úsala como ejemplo público de layout y proceso. No asumas acceso privado y no copies contenido privado o sensible a esta guía.
