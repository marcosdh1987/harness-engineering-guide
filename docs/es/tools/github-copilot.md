# GitHub Copilot

## Rol en un workflow

GitHub Copilot es útil para sugerencias en línea, asistencia por chat y soporte de desarrollo con conocimiento del repositorio.

## Contexto útil

Copilot se beneficia de documentación legible, convenciones estables, ejemplos y una definición clara de la tarea cerca del código o de la documentación que se modifica.

## Adaptación del harness

Un harness debe hacer que el contexto local sea fácil de descubrir, mantener instrucciones duraderas y apoyarse en automatización para verificar sugerencias antes de integrarlas.
