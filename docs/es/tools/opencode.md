# OpenCode

## Rol en un workflow

OpenCode puede actuar como parte de un workflow flexible de asistencia con IA para edición, exploración y ejecución estructurada de tareas.

## Contexto útil

Un harness debe asumir que la herramienta necesita intención clara del repositorio, ejemplos prácticos y acceso sencillo a los pasos de validación.

## Adaptación del harness

Trata a la herramienta como consumidora de reglas compartidas y no como un caso especial. Usa adapters solo cuando cambie la forma de las instrucciones o del empaquetado de la tarea.
