# Claude Code

## Rol en un workflow

Claude Code puede usarse para explorar repositorios, hacer cambios enfocados, validar resultados y apoyar tareas orientadas a revisión.

## Contexto útil

Un harness debe ofrecer reglas claras del repositorio, guías prácticas por tarea, comandos de validación y ejemplos bien ubicados.

## Adaptación del harness

Conviene priorizar instrucciones duraderas y concisas, límites de seguridad explícitos y workflows fáciles de descubrir. Mantén el contexto organizado para que la herramienta pueda seleccionar archivos y verificaciones correctas sin adivinar.
