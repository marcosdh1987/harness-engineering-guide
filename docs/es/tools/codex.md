# Codex

## Rol en un workflow

Las herramientas tipo Codex ayudan con generación de código, edición y ejecución de tareas cuando reciben requisitos claros y pasos de validación.

## Contexto útil

Funcionan mejor cuando el repositorio incluye restricciones explícitas, ejemplos y comandos que definan cómo se ve el éxito.

## Adaptación del harness

Mantén las instrucciones estructuradas, evita objetivos ambiguos y acompaña la generación con validation gates para que los cambios sigan siendo fáciles de revisar.
