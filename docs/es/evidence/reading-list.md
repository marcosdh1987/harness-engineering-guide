# Lista de lectura recomendada

Utilice esta lista de lectura para desarrollar su especialización en Harness Engineering.

## Lecturas esenciales recomendadas

### 1. Adaptadores de instrucciones de herramientas
- **[Claude Code Documentation](https://docs.anthropic.com/en/docs/agents-and-subagents/claude-code)**: Aprenda cómo CLAUDE.md gobierna los permisos de shell locales y los comandos CLI personalizados.
- **[Custom Instructions for GitHub Copilot](https://docs.github.com/en/copilot/configuring-github-copilot/specifying-custom-instructions-for-github-copilot)**: Domine las directrices de Copilot a nivel de repositorio.

### 2. Fundamentos académicos
- **[SWE-agent: Agent-Computer Interfaces](https://arxiv.org/abs/2405.15793)**: Comprenda la importancia crítica de las interfaces Agente-Computadora (ACIs).
- **[AutoDev Framework](https://arxiv.org/abs/2403.08299)**: Estudie el diseño de sandboxes y los pipelines de validación programática.

### 3. Modelado de amenazas y seguridad
- **[OWASP Top 10 for LLM Applications](https://owasp.org/www-project-top-10-for-large-language-model-applications/)**: Lista de verificación de seguridad para vectores de inyección de prompts y exposición de datos.
- **[Security Risks of AI Agents in Software Development](https://snyk.io/blog/security-risks-of-ai-agents-in-software-development/)**: Aprenda sobre inyecciones de prompts indirectas a través de archivos del repositorio.
