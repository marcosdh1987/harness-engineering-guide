# Laboratorio: Validar un cambio

## Objetivo

Practicar cómo validar una edición asistida por IA antes de compartirla.

## Escenario

Una actualización pequeña de documentación parece correcta, pero quieres una verificación de seguridad repetible.

## Pasos

1. Revisa la tarea y los archivos modificados.
2. Ejecuta el comando de validación documentado.
3. Inspecciona el diff para detectar ediciones no relacionadas.
4. Confirma que enlaces, ejemplos y comandos sigan teniendo sentido.

## Resultado esperado

Un cambio revisado con evidencia de que las verificaciones documentadas pasaron.

## Preguntas de reflexión

- ¿Qué verificaciones están automatizadas?
- ¿Qué verificaciones aún requieren revisión humana?

## Notas de seguridad

La validación debe incluir escaneo de secretos y revisión de contenido privado accidental.
