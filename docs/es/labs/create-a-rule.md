# Laboratorio: Crear una regla

## Objetivo

Redactar una regla duradera del repositorio para trabajo asistido por IA.

## Escenario

Tu equipo repite el mismo recordatorio de revisión en cada cambio de documentación.

## Pasos

1. Identifica la expectativa que se repite.
2. Escribe la regla con lenguaje claro y duradero.
3. Colócala donde personas y herramientas puedan encontrarla.
4. Agrega un ejemplo y un paso de validación.

## Resultado esperado

Una regla fácil de reutilizar sin reescribirla en cada prompt.

## Preguntas de reflexión

- ¿La regla es lo suficientemente específica para cambiar comportamiento?
- ¿Es lo bastante amplia para sobrevivir muchas tareas?

## Notas de seguridad

No incluyas secretos, credenciales, URLs privadas ni instrucciones específicas de clientes.
